# iCopy 快速使用教程

iCopy 是一款功能强大的 macOS 剪贴板管理工具，支持文本、图片、文件等多种类型的剪贴板历史记录管理。

## 方式一：下载演示版本

这是最简单快速的使用方式，适合普通用户。

### 步骤

1. **下载应用**
   - 访问项目的 [Releases 页面](https://github.com/YOUR_USERNAME/iCopy/releases)
   - 下载最新版本的 `iCopy.app.zip` 文件

2. **安装应用**
   - 解压下载的 zip 文件
   - 将 `iCopy.app` 拖拽到 `应用程序` 文件夹

3. **首次启动**
   - 双击打开 iCopy
   - 如果遇到"无法打开，因为它来自身份不明的开发者"提示：
     - 打开 `系统设置` → `隐私与安全性`
     - 找到 iCopy 并点击 `仍要打开`
   - 或者在终端执行：
     ```bash
     xattr -cr /Applications/iCopy.app
     ```

4. **授权权限**
   - 首次启动会提示需要辅助功能权限
   - 点击 `打开系统设置` 按钮
   - 在 `隐私与安全性` → `辅助功能` 中勾选 iCopy

## 方式二：从源码编译

这种方式适合开发者或想要自定义功能的用户。

### 前置要求

- macOS 12.0 或更高版本
- Xcode 14.0 或更高版本
- 已安装 Command Line Tools

### 步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/iCopy.git
   cd iCopy
   ```

2. **打开项目**
   ```bash
   open iCopy.xcodeproj
   ```
   或者直接在 Finder 中双击 `iCopy.xcodeproj` 文件

3. **配置签名**
   - 在 Xcode 中选择项目文件
   - 选择 `iCopy` target
   - 在 `Signing & Capabilities` 标签页中
   - 选择你的开发团队（Team）
   - Xcode 会自动配置 Bundle Identifier

4. **编译运行**
   - 点击 Xcode 左上角的运行按钮（▶️）
   - 或使用快捷键 `⌘ + R`
   - 首次编译可能需要几分钟时间

5. **导出应用（可选）**
   - 选择菜单 `Product` → `Archive`
   - 等待归档完成
   - 点击 `Distribute App`
   - 选择 `Copy App` 导出到本地

## 首次使用配置

无论使用哪种方式安装，首次使用都需要进行以下配置：

### 1. 授予辅助功能权限

iCopy 需要辅助功能权限来监听全局快捷键和剪贴板变化。

**配置步骤：**
1. 打开 iCopy 应用
2. 如果弹出权限引导窗口，点击 `打开系统设置`
3. 在系统设置中找到 `隐私与安全性` → `辅助功能`
4. 在列表中找到 iCopy 并勾选
5. 返回 iCopy，点击 `刷新状态` 确认权限已授予

**手动配置：**
- 打开 `系统设置` → `隐私与安全性` → `辅助功能`
- 点击左下角的锁图标解锁
- 点击 `+` 按钮添加 iCopy
- 或在列表中找到 iCopy 并勾选

### 2. 配置快捷键（可选）

在设置页面可以自定义快捷键：

- **提示词选择器**：默认 `⌘ + Shift + P`
- **剪贴板选择器**：默认 `⌘ + Shift + V`
- **显示位置**：可选择顶部、底部、左侧或右侧

### 3. 设置历史记录数量（可选）

- 在设置页面可以配置最大历史记录数量
- 默认保存 100 条记录
- 可根据需要调整为 50、200、500 等

## 基本功能

### 1. 历史剪贴板

iCopy 会自动记录你复制的所有内容，包括：

- **文本**：普通文本、代码片段等
- **图片**：截图、复制的图片文件
- **文件**：文档、压缩包等各类文件
- **文件夹**：目录路径
- **媒体**：音频、视频文件

**使用方法：**
- 点击菜单栏图标选择 `历史剪切板`
- 或使用快捷键 `⌘ + Shift + V` 快速调出选择器
- 支持搜索、筛选、预览、复制和删除操作

### 2. 提示词管理

管理和使用常用的提示词模板，提高工作效率。

**使用方法：**
- 点击菜单栏图标选择 `提示词管理`
- 或使用快捷键 `⌘ + Shift + P` 快速调出选择器
- 可以添加、编辑、删除和分类管理提示词
- 支持搜索和快速复制到剪贴板

### 3. 快速预览

支持 QuickLook 预览功能，快速查看剪贴板内容。

**使用方法：**
- 在历史剪贴板或提示词列表中
- 点击项目的 `详情` 按钮
- 或选中项目后按空格键
- 支持预览文本、图片、文件等多种类型

## 快捷键说明

### 全局快捷键

| 功能 | 默认快捷键 | 说明 |
|------|-----------|------|
| 剪贴板选择器 | `⌘ + Shift + V` | 快速调出剪贴板历史选择器 |
| 提示词选择器 | `⌘ + Shift + P` | 快速调出提示词选择器 |

### 选择器内快捷键

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 搜索 | 直接输入 | 实时搜索过滤内容 |
| 预览 | `Space` | 快速预览选中项目 |
| 复制 | `Enter` | 复制选中项目到剪贴板 |
| 关闭 | `Esc` | 关闭选择器窗口 |
| 上下导航 | `↑` `↓` | 在列表中上下移动 |
